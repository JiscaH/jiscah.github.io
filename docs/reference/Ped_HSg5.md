# Example pedigree: 'HSg5'

A pedigree with five non-overlapping generations and considerable
inbreeding. Each female mated with two random males and each male with
three random females, producing four full-sib offspring per mating. This
is **Pedigree II** in the paper.

## Usage

``` r
data(Ped_HSg5)
```

## Format

A data frame with 1000 rows and 3 variables (id, dam, sire)

# Example genotype file: Griffins

Simulated genotype data from Pedigree
[`Ped_griffin`](https://jiscah.github.io/reference/Ped_griffin.md)

## Usage

``` r
data(Geno_griffin)
```

## Format

A genotype matrix with 142 rows (individuals) and 200 columns (SNPs).
Each SNP is coded as 0/1/2 copies of the reference allele, with -9 for
missing values. Ids are stored as rownames.

# Example genotype file: 'HSg5'

Simulated genotype data for cohorts 1+2 in Pedigree
[`Ped_HSg5`](https://jiscah.github.io/reference/Ped_HSg5.md)

## Usage

``` r
data(SimGeno_example)
```

## Format

A genotype matrix with 214 rows (ids) and 200 columns (SNPs). Each SNP
is coded as 0/1/2 copies of the reference allele, with -9 for missing
values. Ids are stored as rownames.
